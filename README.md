# Java Algorithms

Java implementations and notes for algorithm practice.

## Purpose

This repository is a personal algorithms notebook in code form. It supports interview preparation, implementation fluency, and review of core algorithmic patterns.

## Contents

- Java source code for algorithm problems.
- Practice implementations across data structures and problem-solving patterns.
- Experimental code that can later be refined into katas.

## How To Use

Open the project in IntelliJ IDEA and run individual classes or tests from the IDE.

## Status

Active learning repo. The next useful upgrade is to organize problems by topic and add short notes for complexity, invariant, and edge cases.

## Validation

Add sample cases or tests for each algorithm, including edge cases and complexity-sensitive input shapes.

## Engineering Notes

The strongest signal in this repo is clarity of reasoning: problem statement, invariant, complexity, and implementation should stay close together.
